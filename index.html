BrewBuddy

Connecting Offices through Smart Motion Detection and Video Streaming

About the Project

BrewBuddy is an innovative solution designed to enhance connectivity and collaboration between two office spaces. The project utilizes motion detection sensors, Raspberry Pi devices, and Agora's video conferencing technology to enable seamless, automatic video streaming whenever activity is detected in both locations.

Key Features

  • Motion detection sensors for activity monitoring in office spaces
  • Automatic video streaming between two offices using Raspberry Pi and screens
  • Video conferencing powered by Agora for real-time communication
  • Compact and efficient implementation using Raspberry Pi Pico W

Technology Stack

  • Raspberry Pi and Raspberry Pi Pico W
  • Agora Video SDK for seamless video conferencing
  • Motion detection sensors
  • Custom web server for project management

How It Works

When motion is detected in both office spaces, the system automatically initiates a video stream between the two locations. The sensors monitor activity, and the Raspberry Pi devices handle the communication, ensuring real-time interaction.

Future Enhancements

  • Integration with more advanced sensors for enhanced accuracy
  • Support for additional offices and scalability
  • Improved user interface for easier management
